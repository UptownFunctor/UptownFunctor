Delimiter Remover

This application allows the user to remove specific delimiters from their text. Simply paste the text you wish to use into the textbox to the right, and enter the delimiter you wish to remove from the text into the input box on the right.

The delimiter string can be any valid unicode string, including spaces. Every character in the delimiter string will be deleted from input text.
